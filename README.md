# 한국 주식시장 거래량 폭발 임계치 백테스팅 프로젝트

## 프로젝트 개요

한국 주식시장(KOSPI, KOSDAQ)에서 거래량이 급증하는 시점을 포착하여 수익성을 분석하는 백테스팅 프로젝트입니다.

## 목적

거래량 폭발을 감지하는 최적의 임계치를 찾아내고, 해당 시그널 발생 후 수익률 패턴을 분석합니다.

## 데이터

- **대상 종목**: KOSPI 시가총액 상위 200개 + KOSDAQ 시가총액 상위 150개
- **기간**: 2022년 1월 ~ 2024년 12월 (3년)
- **데이터**: 일봉 (시가/고가/저가/종가/거래량)
- **라이브러리**: FinanceDataReader 또는 pykrx

## 분석 지표

### 1. Volume_Ratio (거래량 비율)
```
Volume_Ratio = 당일 거래량 / 20일 평균 거래량
테스트 값: 2.0, 3.0, 4.0, 5.0배
```

### 2. Z_Score (거래량 Z-스코어)
```
Z_Score = (당일 거래량 - 60일 평균) / 60일 표준편차
테스트 값: 1.5, 2.0, 2.5, 3.0
```

## 그리드 서치

- Volume_Ratio와 Z_Score의 모든 조합 테스트 (4 × 4 = 16개 조합)
- 각 조합별 수익률 측정:
  - 익일 수익률
  - 3일 후 수익률
  - 5일 후 수익률
  - 10일 후 수익률

## 출력 결과

### 1. 통계 데이터 (CSV)
- 조합별 시그널 발생 빈도
- 평균 수익률, 승률, 중위값
- 샤프지수
- 최대 수익/손실

### 2. 시각화 (PNG)
- 히트맵: 조합별 평균 수익률
- 박스플롯: 수익률 분포
- 막대 그래프: 시그널 발생 빈도
